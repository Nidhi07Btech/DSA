# Data Structures and Algorithms (DSA) Practice

Welcome to my DSA repository! In this project, I am practicing and solving core Data Structures and Algorithms problems to strengthen my programming logic and problem-solving skills. 

Up until now, I have covered the fundamental concepts of programming and solved multiple problems based on them.

---

## 🚀 Concepts Covered So Far

### 1. Variables and Operations
Basic building blocks of programming, focusing on data storage and basic manipulations.
* **Addition of 2 Variables**
* **Swap Two Variables ($a$ and $b$)**
* **Variable Operations**

### 2. Conditional Statements
Logic-building using conditions to control the flow of the program.
* **If statement**, **If-Else statements**, and **If-Else-If ladder**
* **Largest among two numbers** & **Largest among three numbers**
* **Switch case** applications
* **Ternary Operator** examples

### 3. Loops (While Loop & For Loop) and Control Statements
Mastering repetition, iterations, and controlling loops using statements like `break`.
* **Print N Even numbers**
* **Print the Series** & **Series using 5 multiples**
* **Print digits of a number**
* **Reverse a number**
* **Armstrong numbers**
